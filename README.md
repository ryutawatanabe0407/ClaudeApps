# ClaudeApps 🎨

## グラデーションジェネレーター

美しいCSSグラデーションを簡単に作成できるWebアプリケーションです。

### 機能

- 🎨 インタラクティブなカラーピッカー
- 🔄 グラデーションの向きと角度の調整
- 📋 ワンクリックでCSSコードをコピー
- 🎭 プリセットカラー
- ⌨️ ランダムグラデーション生成（Rキー）
- 📱 レスポンシブデザイン

### デモ

GitHub Pagesでデプロイされたアプリを確認できます。

### 技術スタック

- HTML5
- CSS3 (Animations, Grid, Flexbox)
- Vanilla JavaScript
- GitHub Actions (自動デプロイ)

### ローカルでの実行

```bash
# リポジトリをクローン
git clone <repository-url>

# ブラウザで index.html を開く
open index.html
```

### デプロイ

mainブランチまたは claude/github-pages-deployment-80rVt ブランチにプッシュすると、GitHub Actionsが自動的にGitHub Pagesにデプロイします。
