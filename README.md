# ClaudeApps

## ガントチャートツール

プロジェクトのタスクを視覚的に管理できるWebアプリケーションです。

### 機能

- タスクの追加・編集・削除
- ガントチャートでの視覚的なタスク管理
- 進捗率の設定と表示
- Excelファイルのインポート・エクスポート
- 日/週/月単位での表示切り替え
- レスポンシブデザイン

### 技術スタック

- HTML5
- CSS3
- Vanilla JavaScript
- SheetJS (xlsx)

### ローカルでの実行

```bash
# ブラウザで gantt.html を開く
open gantt.html
```
