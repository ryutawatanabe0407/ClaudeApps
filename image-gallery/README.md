# Image Gallery

シンプルで使いやすい画像ギャラリーアプリケーション。フォルダから画像を読み込み、グリッド表示、タイトル編集、並べ替え、連番化などの機能を備えています。

## 特徴

- **フォルダ読み込み**: フォルダを選択して一括で画像を読み込み
- **グリッド表示**: 列数と行数を自由に調整可能
- **タイトル編集**: 各画像にタイトルを設定し、編集可能
- **ドラッグ&ドロップ**: 画像をドラッグして簡単に並べ替え
- **連番化**: 一括で画像に連番をつける機能
- **ミニマルデザイン**: MUJI・IKEA風のシンプルでオシャレなデザイン

## 使い方

1. `index.html`をブラウザで開く
2. 「フォルダを選択」ボタンをクリックして画像フォルダを選択
3. 列数と行数を調整してグリッドレイアウトを変更
4. 各画像のタイトルをクリックして編集
5. 画像をドラッグ&ドロップで並べ替え
6. 「連番化」ボタンで一括連番付け

## 技術スタック

- HTML5
- CSS3
- Vanilla JavaScript

## 機能詳細

### フォルダ読み込み
- 画像ファイル（PNG, JPG, GIF など）を自動検出
- ファイル名を初期タイトルとして設定

### グリッド表示
- 1〜10列、1〜10行まで設定可能
- レスポンシブ対応

### タイトル編集
- 各画像のタイトルを個別に編集可能
- リアルタイムで変更を反映

### ドラッグ&ドロップ
- 画像をドラッグして位置を入れ替え
- ビジュアルフィードバック付き

### 連番化
- 接頭辞と開始番号を指定
- 3桁のゼロパディング（例: Image001, Image002...）
